## JD-001：AI Agent 工程师｜益海嘉里丰益信息

### 基本信息

- 岗位：AI Agent 工程师。
- 公司 / 厂商：益海嘉里丰益信息。
- 联系人截图信息：侯先生，益海嘉里丰益信息，HR。
- 工作地址：无锡滨湖区平安财富中心益海嘉里。
- 地点：无锡。
- 薪资：10-15K。
- 年限：1-3 年。
- 学历：本科。
- 岗位标签：机器学习类 AI、B 端产品。

### JD 摘要

岗位职责：

1. 参与需求拆解、技术方案评审和项目排期，推动 AI Agent 产品从原型到上线的全流程落地。
2. 跟踪并调研 Agent 领域前沿技术，例如多模态交互、工具调用、记忆增强、自主规划，探索并落地技术优化方案。
3. 持续关注大模型与 AI Agent 生态，把行业实践和创新思路融入日常开发，为产品迭代和技术突破提供支持。

任职要求：

1. 本科及以上，计算机、软件工程、人工智能、数据科学、数学统计等相关专业。
2. 具备良好的沟通表达和跨团队协作能力，能清晰传达技术方案与风险，高效推进项目。

### 匹配判断

匹配点：

- 用户有 Java 后端开发基础，可以切 AI 应用工程方向。
- `interview-assistant` 可改造成 AI 面试准备 Agent，作为岗位作品。
- 现有学习资料已覆盖 LLM 接入、Prompt、结构化输出、RAG、幻觉处理、降级和成本控制。

短板：

- Tool Calling、Memory、Planning 还没有项目化落地。
- 还没有一份完整的 Agent 技术方案文档和流程图。
- 真实工作经历主要是银行业务开发，AI Agent 目前更多是学习和项目成长线。

### 简历定制方向

简历主线：

```text
Java 后端开发 + AI 应用工程化学习与项目实践
```

推荐突出：

- `interview-assistant` 智能面试助手项目。
- Spring AI、Prompt 模板、结构化输出、AI 失败兜底。
- 后续 Agent 化计划：Tool Calling、RAG、用户薄弱点记忆、学习计划生成。
- 技术方案表达能力：需求拆解、风险说明、任务切片。

避免夸大：

- 不写“负责完整 AI Agent 平台上线”。
- 不写“精通多模态 / 多 Agent / LangGraph”，除非后续真的完成。
- 不把 `interview-guide` 写成个人项目。

### 基本信息

- 岗位：Java 开发工程师。
- 公司 / 用工方：无锡锡商银行。
- 联系人截图信息：沈佳栋，无锡锡商银行，人事经理。
- 地点：无锡锡山区红豆东方财富广场西区 9 楼。
- 薪资：12-24K。
- 年限：3-5 年。
- 学历：本科。
- 技术标签：Java、Spring Cloud、MySQL、Oracle。

### JD 摘要

岗位职责：

1. 负责完成相关系统和相关核心模块的代码编写和实现。
2. 负责完善系统相关技术文档的沉淀和总结。
3. 负责解决开发过程中的技术瓶颈和问题。
4. 负责系统日常技术支持和系统维护。
5. 负责带领厂商开发工程师完成系统研发工作。
6. 完成领导交办的其他工作。

岗位要求：

1. 本科及以上，信息技术管理等相关专业优先。
2. 熟练掌握 Java，了解 Go、Python、JavaScript 中至少一门，熟悉 Linux。
3. 熟练掌握 Spring、Spring MVC、MyBatis/Hibernate 等框架，理解 MVC 和 B/S 架构。
4. 熟悉 Spring Cloud、Dubbo 等微服务框架，理解服务治理和调优。
5. 熟悉 MySQL、Oracle、MongoDB、Redis 中至少两种数据库，有性能调优和数据库系统设计经验。
6. 熟悉分布式、高并发、高可用架构，具备海量业务架构、研发、运维、优化经验。
7. 了解工作流、规则引擎、MQ、Lucene/Solr/ElasticSearch、Jenkins、Sonar 等中间件和工具。
8. 有对公信贷产品设计和项目管理经验者优先。

- 2024 年毕业。

- 2026-06 开始在宁波银行驻场外包，与同事参与数币业务开发。
- 在长亮主要做业务方面开发，包括长亮 V8.7 基板的开户业务、存入业务、支取业务等交易。

匹配点：

- 有真实银行 / 金融科技开发经历。
- 做过存款组交易类业务，和银行核心系统、账户、存款、交易链路高度相关。
- 当前接触宁波银行数币业务，对银行现场交付和联调有现实经历。
- 可以突出业务理解、交易开发、问题排查、文档沉淀和协作能力。

短板：

## 通用简历定制规则

### 投银行 Java 岗

优先突出：

- 长亮科技存款组真实经历。
- 开户、存入、支取交易链路。
- 宁波银行数币驻场。
- 交易一致性、幂等、流水、异常处理。
- Spring / MyBatis / SQL / Linux / Redis / MQ。

项目排序建议：

1. 长亮科技 / 银行存款组工作经历。
2. 宁波银行数币驻场经历。
3. `interview-assistant` 作为 AI + Java 学习项目补充。

### 投 AI Agent 岗

优先突出：

- `interview-assistant` Agent 化改造。
- LLM 接入、Prompt、结构化输出、失败兜底。
- Tool Calling、Memory、RAG、Planning 的学习和落地计划。
- Java 后端稳定性、接口设计、任务状态和日志。

项目排序建议：

1. `interview-assistant` / AI 面试准备 Agent。
2. 银行业务 Java 开发经历，证明后端工程能力。
3. 其他学习项目。

### 投普通 Java 后端岗

优先突出：

- Java 基础、Spring、MyBatis、MySQL、Redis。
- 银行交易开发经历。
- 问题排查和联调经验。
- `interview-assistant` 作为 Spring Boot + AI 项目亮点。
