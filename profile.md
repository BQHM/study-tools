## 3. 求职主线

### 主线：银行 Java / 金融科技 Java

定位：

```text
银行核心业务 Java 开发工程师，具备存款账户、开户、存入、支取等交易开发经验，当前参与数字人民币业务开发。
```

适合投递：

- 银行科技 Java 开发。
- 金融科技 Java 开发。
- 银行外包核心系统开发。
- 存款、账户、支付、数币、信贷相关 Java 岗。
- 要求 2-4 年或 3 年左右的 Java 岗。

优势：

- 有真实银行业务开发经历。
- 有长亮科技存款组经历。
- 做过开户、存入、支取这类银行交易。
- 有产品研发背景，并参与过多个银行客户 POC / 驻场支持。
- 当前有宁波银行数币驻场经历。
- 比纯学习项目候选人更有业务可信度。

短板：

- 工作年限约 2 年多，部分 3-5 年 JD 会略低。
- 微服务治理、Spring Cloud、Dubbo、Redis、MQ、ES、Jenkins、Sonar 需要补强。
- 高并发、高可用、海量业务架构不能包装成主导经验。
- 对公信贷产品设计和项目管理经验目前不足。

### 辅线：AI Agent / Java + AI 应用工程化

定位：

```text
Java 后端开发 + AI 应用工程化学习与项目实践。
```

适合投递：

- AI Agent 工程师。
- Java + AI 应用开发。
- LLM 应用工程。
- B 端 AI 产品后端开发。

优势：

- 有 Java 后端工程背景。
- 已开始系统学习 Spring AI、Prompt、结构化输出、Tool Calling、RAG、Memory、Planning。

短板：

- AI Agent 更多是学习和项目成长线，不是当前真实工作主线。
- Tool Calling、Memory、Planning、RAG 还没有完整落地。
- 不能把 AI 辅助项目包装成完全独立商业项目。
